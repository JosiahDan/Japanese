# なくてもいいです  

不必做某事、不用做某事

1. **動詞~~ない~~ + なくてもいいです/なくてもかまいません**
2. **い形く~~ない~~ + なくてもいいです/なくてもかまいません**
3. **な形じゃ~~ない~~ + なくてもいいです/なくてもかまいません**
4. **名詞じゃ~~ない~~ + なくてもいいです/なくてもかまいません**  
***
* 例:  
1. 明日は休みですから、早く起きなくてもいいです。  
因为明天放假，不早起也可以。
2. ここでは靴を脱ぐなくてもいいです。  
这里不脱鞋也可以。
3. 週末は外食しますから、料理を作らなくてもいいです。  
周末要在外边吃饭，可以不用做饭。
4. この件は部長に報告しなくてもかまいませんか。  
这件事可以不向经理报告吗？
