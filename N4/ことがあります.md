# たことがあります  
```
動詞た形 + ことがあります。 有做过...
動詞辞書形 + ことがあります。 有时、偶尔会出现某种情况。 
動詞ない形 + ことがあります。 有时、偶尔不会出现某种情况。
```
* 例:  
1. 富士山に登ったことがあります。　　
我有爬过富士山。  
2. おばけを見たことがありますか。  
你有见过妖怪吗。
3. 外国へ行ったことがありません。　　
我曾经去过外国。
4. 今まで女性と付き合ったことがありません。  
我至今没有和女性交往过。  
5. このパソコンは突然止まることあります。  
这个电脑有时会突然停止工作。  
6. ツアーは参加人数が少ないとき、中止になることがあります。　　
游乐场参见的人数很少的时候，有时会停办。
7. 空港バスは渋滞で遅れることがあるから、早めに出発しよう。  
机场的巴士有时候会因为堵车延迟，所以早点出发吧。
