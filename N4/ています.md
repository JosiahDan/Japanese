# ています
```
動詞て形います
```
## 用法解释  
1. **现在进行时**  
例:  
(1)ご飯を食べています。  
现在正在吃饭。
2. **表示目前的状态**  
例:  
(1)椅子に座っています。  
在椅子上坐着。  
(2)大阪に住んでいます。  
在大阪住。  
3. **表示习惯**  
毎日日記を書いています。  
## 例文  
1. 今、出口先生の動画を見て日本語を勉強しています。  
现在我在看出口老师的影片学习日语。  
2. 私は鎌倉に住んでいます。  
我现在在镰仓住。(表示状态)
3. トイレの電気がついていますから、消してください。  
厕所的灯亮着，请把它关掉。
4. 毎晩日記を書いています。  
每晚都写日记(表示状态)。
