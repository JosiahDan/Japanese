# うとします　　
```
動詞意向形 + とします  
```
## 释义  
正要做某事，常与`とき`连用表示正要做某事的时候。  
1. Aうとしたとき、B。(正要做A时候，又去做了B)  
きっぷを買おうとした時、財布を忘れたことに気がつきました。  

2. Aうとしません。表示在大家都期望某人做A时并没有做A  
息子は野菜を食べようとしません。  
## 例文  
1. 出かけようとしたとき、友達から電話がかかって来ました。  
正要出门的时候朋友打过来了电话。  
2. たくさんの人がマスクを買おうとして、店の前に並んでいます。　　
许多人想要购买口罩的时候在店门口排队。  
3. 犯人は警察の質問に答えようとしなかった。　　
犯人不肯回答警察的问题。
