# よにしてください  
```
動詞辞書形/ない形 + ようにしてください
```
## 释义  
请尽量做某事，相比直接使用`ください`更有拒绝的余地，是一种建议而不是要求。`ください`是一种礼貌的要求。    
## 例文
1. 毎日運動するようにしてください。  
请尽量每天都运动。  
2. お酒を飲まないようにしてください。  
请尽量不要喝酒。  
3. なるべく前のほうに座るようにしてください。  
请尽量在固定的时间睡觉。  
4. 決まった時間に寝るようにしてください。  
请尽量在固定的时间睡觉。  
5. 健康のためにタバコは吸わないようにしてください。  
为了健康，请尽量不要抽烟。
